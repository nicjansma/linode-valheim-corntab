# linode-valheim-corntab 🌽

Launches a [Linode](https://www.linode.com/) to run a dedicated [Valheim](https://www.valheimgame.com/) game server.

Helps us play on powerful, dedicated Linode instances (e.g. 64 GB and 32 CPU) for on-demand (or on a weekly schedule) play sessions.

This application was written with AI assistance (Claude Opus 4.5) for code generation, iterative feature development, and review.
